# 🎨 Asset Monitoring System - Frontend Client

This directory contains the React + TypeScript frontend client for the **Asset Monitoring & Management System**, built with Vite and Tailwind CSS.

For full project details, database setup, and backend instructions, please refer to the [Root README](../README.md).

## 🚀 Key Libraries & Features
*   **React 19 & Vite**: Fast Refresh and hot module replacement.
*   **Tailwind CSS v4**: Modern utility-first styling.
*   **Radix UI**: Accessible primitive components.
*   **Recharts**: Interactive data visualizations for the Dashboard.
*   **Framer Motion**: Smooth micro-animations and transitions.

## 🛠️ Development Scripts

From this directory, you can run the following commands:

### `npm run dev`
Starts the Vite development server locally:
```bash
npm run dev
```
By default, the server runs on `http://localhost:5173`.

### `npm run build`
Compiles TypeScript and bundles the assets for production:
```bash
npm run build
```
This generates optimized static files in the `dist/` folder.

### `npm run lint`
Runs ESLint to analyze static code quality and style constraints:
```bash
npm run lint
```
